## Getting started

### Prerequisites

- C++ compiler supported C++20
- Conan (optional)

### Build

It is a header-only library.

### Install (using Conan)

Add repository:

```shell
conan remote add isnullxbh/testing https://109.195.38.72:8081/artifactory/api/conan/public-conan
```
